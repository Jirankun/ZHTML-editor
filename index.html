Panduan Penggunaan Editor Kode

1. Menulis Kode

Gunakan editor utama untuk menulis kode HTML, CSS, atau JavaScript. Editor ini mendukung syntax highlighting dan auto-completion.

2. Pratinjau Langsung

Klik tombol "Pratinjau" untuk melihat hasil kode Anda secara real-time dalam modal popup.

3. Mode Split View

Klik ikon split di header untuk membuka mode layar terbelah yang menampilkan editor dan pratinjau secara bersamaan.

4. Salin Kode

Gunakan tombol salin untuk menyalin seluruh kode ke clipboard.

5. Bar Simbol Cepat

Gunakan tombol-tombol di bagian bawah layar untuk memasukkan simbol-simbol umum seperti kurung, tanda kutip, dll.

6. Penyimpanan Otomatis

Kode Anda akan otomatis disimpan ke penyimpanan lokal aplikasi.

7. Keamanan

Aplikasi ini menggunakan Content Security Policy dan sanitasi kode untuk memastikan keamanan saat menjalankan pratinjau.

8. Dukungan Link Eksternal

Link eksternal dalam pratinjau akan dibuka dalam modal aman untuk mencegah navigasi keluar aplikasi.

⚠️ Peringatan Keamanan

Meskipun aplikasi ini dirancang dengan standar keamanan tinggi, selalu berhati-hati dengan kode yang Anda jalankan. Jangan menjalankan kode dari sumber yang tidak terpercaya.
